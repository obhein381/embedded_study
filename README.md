# Embedded Study

A collection of hands-on embedded systems projects and learning notes using the ESP32.

## Goals

- Learn embedded systems from the basics.
- Understand how hardware and software interact.
- Build a solid foundation for a smart farm project.

## Learning Roadmap

- [x] 01_LED
- [ ] 02_Button
- [ ] 03_Debouncing
- [ ] 04_Serial
- [ ] 05_Soil Moisture Sensor
- [ ] 06_Relay
- [ ] 07_Water Pump
- [ ] Mini Smart Farm

## Development Environment

- Board: ESP32 Dev Module (ESP32-WROOM-32)
- Framework: Arduino
- IDE: VS Code + PlatformIO
